# JavaScript Expense Calculator

A simple and modern expense calculator built with "Vanilla JavaScript".

## Features
- Add expenses dynamically
- Calculate total expense using `Array.reduce()`
- Real-time DOM update
- Input validation with alert
- Clean & modern UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository  
2. Open `index.html` in your browser  

## Purpose
This project is created for practicing JavaScript "array methods", "DOM manipulation", and "basic validation".
